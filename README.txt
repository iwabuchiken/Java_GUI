	Greetings

***************************
Welcome to Java_Gui application. This is a simulation tool
	to see how different placement patterns will look like
	when executed.

May not be so easy to use, but I find it very simple. Enjoy!

***************************

	Usage

***************************
1. Button "Execute"

	1) Calculate the residues in each placement pattern
	
2. Button "Clear"

	1) Reset the entire canvas: i.e. Set the statuses of
		the rectangle C and B to the default value of 1

3. Menu bar item "Options" --> Drop down item "Reload"

	1) Reload the data of the rectangles A, B and C
		from the property file "config.properties" in "data" folder
